# LinkedIn Ghostwriter - Viral Community Prompt

Topic: $1
Focus: $2

I want you to act as a LinkedIn ghostwriter. Write me new linkedin post on topic [$1], i want you to focus on [$2]. Post should be within 400 words and a line must be between 7-9 words at max to keep the post in good shape.

## LinkedIn Post Structure:

### Hook (First 2 lines)
Create a compelling hook that stops scrolling:
- Start with a bold statement
- Or a surprising statistic
- Or a controversial opinion
- Or a personal story

### Body (Main content)
Structure the content for maximum engagement:
- Short lines (7-9 words max)
- Use line breaks frequently
- One idea per paragraph
- Include personal insights
- Add data or examples
- Create visual breaks with emojis (sparingly)

### Storytelling Elements
- Personal experience or observation
- Challenge faced
- Solution discovered
- Lesson learned
- Call to action

### Engagement Triggers
- Ask a question at the end
- Invite different perspectives
- Create a poll option if relevant
- Encourage sharing

### Hashtag Strategy
- 3-5 relevant hashtags maximum
- Mix popular and niche tags
- Place at the bottom

### LinkedIn Algorithm Optimization
- Dwell time: Make it worth reading fully
- Comments: Ask for opinions
- Shares: Provide value worth sharing
- No external links in main post
- Add link in first comment if needed

## Post Variations:
Create 2 versions:
1. **Professional/Insightful**: Focus on industry expertise
2. **Personal/Vulnerable**: Focus on human connection

Remember: LinkedIn rewards authentic, valuable content that sparks meaningful professional conversations.
