## O Essencial em 5 Minutos

### O que são agentes?
Versões especializadas do Claude. Como ter um expert de cada área disponível 24/7.

### Como usar?
1. Escolha o agente certo para a tarefa
2. Siga o workflow: Research → Planning → Implementation → Testing
3. Use comandos "think" para análises complexas
4. Sempre teste o código gerado

### Qual agente escolher?
- **Criando projeto?** → `backend-architect`
- **Escrevendo Python?** → `python-pro`
- **Fazendo frontend?** → `frontend-developer`
- **Procurando bugs?** → `security-auditor`
- **Configurando deploy?** → `deployment-engineer`

### Exemplo rápido
```
Com python-pro:
"crie uma função para calcular desconto com validação e testes"

Você recebe:
- Função com type hints
- Validação completa
- Tratamento de erros
- Testes unitários
- Documentação
```

## Dicas de Ouro

1. **Não pule o planning** - Sempre pesquise e planeje antes
2. **Seja específico** - Quanto mais detalhes, melhor o resultado
3. **Peça explicações** - "por que fez assim?"
4. **Compare com seu código** - Melhor forma de aprender
5. **Teste tudo** - Nunca confie cegamente

## Evolução Natural

### Semana 1
- Use um agente simples
- Faça tarefas pequenas
- Compare resultados

### Semana 2
- Combine 2-3 agentes
- Projetos maiores
- Entenda o workflow

### Mês 1
- Use todos os agentes relevantes
- Crie projetos completos
- Desenvolva seu estilo

### Depois
- Crie seus próprios agentes
- Otimize workflows
- Ensine outros

## Erros Comuns

- **Começar muito grande** - Faça coisas simples primeiro
- **Não testar** - Sempre valide o código
- **Aceitar sem entender** - Questione e aprenda
- **Usar agente errado** - Backend para backend, frontend para frontend

## Precisa de Ajuda?

1. Releia os exemplos
2. Tente algo mais simples
3. Seja mais específico
4. Peça explicações ao agente

Lembre-se: todo expert foi iniciante. A diferença é a prática.
