## Quick Selection Guide

**Use Stage 1 when:**
- Just starting to collect notes on a theme
- Have 7-10 related notes
- Need quick organization
- Want to see what you have before structuring

**Use Stage 2 when:**
- Actively working on a project/theme
- Developing understanding through writing
- Need workbench for exploration
- Discovering sub-themes and patterns

**Use Stage 3 when:**
- Theme has matured and stabilized
- Want reliable reference
- Synthesized deep understanding
- Creating hub for future work

**Use Effort/Project when:**
- Active project with clear outcome
- Time-bound initiative
- Need action tracking
- Temporary focus (will archive when done)

**Use Index.md when:**
- Organizing entire folder
- Creating navigation for section
- Want persistent structure
- Folder is stable part of vault
